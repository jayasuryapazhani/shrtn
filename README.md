# Shrtn

Shrtn is a Chrome extension that shortens web links and generates shareable QR codes.

## Planned technology stack

- React
- Vite
- Chrome Manifest V3
- Node.js
- Express.js
- PostgreSQL

## Project structure

- `extension` — React Chrome extension
- `server` — Node.js and Express backend
- `docs` — architecture and API documentation
- `postman` — Postman collections
- `tests` — shared integration and end-to-end tests
