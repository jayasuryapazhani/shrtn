Shrtn Privacy Policy

Last updated: July 15, 2026

Overview

Shrtn is a browser extension and URL-shortening service. It creates short links, generates QR codes, and displays redirect analytics.

Information processed

When a user opens the Shrtn extension, it may read the URL of the active browser tab to prefill the URL field.

When the user chooses to shorten a URL, that URL is sent securely to the Shrtn API.

The Shrtn service stores the original URL, generated short code, creation timestamp, redirect click count, and most recent click timestamp.

How information is used

This information is used only to create and operate short links, generate QR codes, perform redirects, and provide link analytics.

Information not collected

Shrtn does not require an account and does not intentionally collect names, email addresses, payment information, passwords, personal communications, or page content.

Sharing and selling

Shrtn does not sell user data. Data is not used for advertising, credit decisions, or unrelated purposes.

Infrastructure providers may process information only as required to host the Shrtn API and PostgreSQL database.

Security

Data is transmitted using HTTPS. Shrtn applies request validation, security headers, request-size limits, and rate limiting.

Data retention

Short-link records are retained so that generated links continue to redirect and provide analytics.

User choices

Shrtn reads the active tab URL only when the user invokes the extension. Users can also replace the detected URL before creating a short link.

Contact

Privacy questions can be submitted through the Shrtn GitHub repository.
