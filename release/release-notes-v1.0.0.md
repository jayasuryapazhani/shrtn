# Shrtn v1.0.0

The first stable release of Shrtn.

## Highlights

- Public URL-shortening website
- Chrome and Brave Manifest V3 extension
- Active-tab URL detection
- Persistent PostgreSQL-backed short links
- Public redirect URLs
- Click analytics
- QR-code generation and download
- Copy and open actions
- Railway production deployment
- Responsive sketch-style website and extension UI
- Teal and cream extension icon set
- API validation, security middleware, and rate limiting
- Automated backend and extension tests

## Technology

- React
- Vite
- Node.js
- Express
- PostgreSQL
- Neon
- Railway
- Vitest
- Supertest
- GitHub Actions

## Installation

1. Download `shrtn-extension-v1.0.0.zip`.
2. Extract the ZIP.
3. Open `chrome://extensions` or `brave://extensions`.
4. Enable Developer mode.
5. Select Load unpacked.
6. Choose the extracted extension directory.
